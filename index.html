¿Cuales son mis habilidades?

Soy desarrollador web recibido de CoderHouse y tengo conocimientos en HTML,CSS,SASS,CEO y actualmente me encuentro capacitandome en JAVASCRIPT

¿Que es HTML:5?

Toda pagina web cuenta con este tipo de estructura basica que es el esqueleto de una pagina web para luego aplicar los estilos y darle vida a la pagina

¿Que es CSS3?

El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. Es decir, permite generar el diseño visual de páginas web e interfaces de usuario..

¿Que es JavaScript?

El JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás.(actualmente me encuentro aprendiendo este lenguaje).
